Cupcake Builder Game


A game where a user must click on specific cupcake elements cascading down the screen. The elements are: a base, ice cream, frosting and topping. When one of each element is chosen, the cupcake is complete, as is the game. If a user clicks on more than one of the same element, they lose a cupcake life. Lose three lives and the game is over. Created with a lot of Javascript and jQuery.

Trip Planner for World Cup 2014

A Ruby on Rails application for the 2014 World Cup in Brazil. Users can view the complete game schedule and view detailed inforomation about host cities and stadiums. The app also includes a basic trip planner to allow a user to see games and locations for specific dates.
