Once Upon a Cupcake E-commerce Site

A Rails 4.0 e-commerce application for an actual business that makes and sells ice cream cupcakes. Users can add items to a shopping cart and checkout using the Stripe gem. The site includes an interactive custom flavor builder where a user can create their own flavor and then purchase it as a custom order. Users and admins also receive emails upon order submission.

Cupcake Builder Game


A game where a user must click on specific cupcake elements cascading down the screen. The elements are: a base, ice cream, frosting and topping. When one of each element is chosen, the cupcake is complete, as is the game. If a user clicks on more than one of the same element, they lose a cupcake life. Lose three lives and the game is over. Created with a lot of Javascript and jQuery. This game is included in the Once Upon a Cupcake e-commerce application.
